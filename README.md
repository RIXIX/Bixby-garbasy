# 삼성 빅스비(Samsung Bixby) 캡슐 개발

## 서비스 명 : 쓰레기 박사

### 삼성 빅스비 개발 대회 수상작 - 장려상


빅스비를 통해 헷갈리는 분리수거 쓰레기 정보를 제공하고, 현 위치를 통한 버리는 장소, 동네에 따른 다른 버리는 요일을 안내합니다.

데이터 : 공공데이터포탈(전국쓰레기데이터API) 및 자체 DB(쓰레기 데이터)-환경부 출처 구축
